Module rshark::ip [−] [src]

Dissection of Internet Protocol (IP) packets.

This module will eventually contain dissectors for protocols in the IP suite, e.g., rshark::ip::icmp and rshark::ip::tcp. For now, it only handles IP headers.

See RFC 791.
